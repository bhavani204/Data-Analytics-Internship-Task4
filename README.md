# 📊 Sales Performance Analysis using Power BI
## Task 4 – Data Storytelling & Statistical Validation

## 📌 Project Overview

This project focuses on analyzing sales performance using Microsoft Power BI and transforming raw business data into meaningful insights through data storytelling. Interactive dashboards were created to visualize sales trends, product performance, regional distribution, and key business metrics. Statistical validation was also incorporated to support data-driven decision-making.

---

## 🎯 Project Objectives

- Analyze overall sales performance using Power BI.
- Create interactive dashboards for business visualization.
- Monitor key performance indicators (KPIs).
- Identify sales trends across products, categories, and regions.
- Perform data storytelling to communicate business insights.
- Validate findings using statistical analysis.
- Provide actionable business recommendations.

## 📊 Statistical Validation

Business Question:

**Is there a significant relationship between Sales and Profit?**

### Null Hypothesis (H₀)

There is no significant relationship between Sales and Profit.

### Alternative Hypothesis (H₁)

There is a significant relationship between Sales and Profit.

### Statistical Method

- Pearson Correlation Analysis
- Confidence Level: 95%
- Significance Level (α = 0.05)

The statistical analysis validates the dashboard insights and supports evidence-based business decisions.
